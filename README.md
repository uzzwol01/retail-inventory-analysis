# Retail Inventory Analysis

## Overview
This project analyzes retail sales and inventory data to identify trends,
improve stock management, and support business decisions.

## Tools
- SQL
- Python (Pandas, NumPy)
- Tableau

## Dataset
Source: Kaggle Retail Sales Dataset

## Objectives
- Analyze sales trends
- Identify top products
- Evaluate inventory performance
- Build dashboards

## Status
Work in progress
